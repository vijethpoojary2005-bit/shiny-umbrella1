\# Smart File Organizer



Smart File Organizer is a desktop application that automatically organizes files into folders based on type or date.



\## Features

\- Organize files by type

\- Organize files by date

\- User-friendly interface

\- Desktop executable (.exe)



\## Technologies Used

\- Python

\- Tkinter

\- OS module

\- Shutil module



\## How to Run

1\. Install Python

2\. Run:

&nbsp;  python file\_organizer\_app.py
